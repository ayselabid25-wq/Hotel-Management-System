# Hotel-Management-System
This project is a Hotel Management System developed as part of my Database Systems course. It is designed to efficiently manage hotel operations, including guest records, room bookings, staff management, billing, and payments. The project demonstrates the practical implementation of database design concepts using SQL Server.
features:
• Guest Registration and Management
• Room Booking and Reservation
• Staff Management
• Billing and Payment Management
• Room Availability Tracking
• SQL Queries for Data Retrieval
• Stored Procedures
• User-Defined Functions
• Database Triggers
• Database Normalization (Up to 3NF)
• Entity Relationship Diagram (ERD)
Skills Demonstrated:
• Database Design
• SQL Programming
• ERD Design
• Database Normalization
• Stored Procedures
• Triggers
• Functions
• Problem Solving
